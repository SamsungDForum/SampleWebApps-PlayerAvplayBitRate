This App demonstrates how to controll adaptive stream with the use of AVPlay.

User can switch streams playing, clicking the button under Current Video label, and choosing aproperiate one.

User also can change which file from the adaptive stream manifest, should play, under the Video label

User can add own streams, adding objects to streamData array in config.js file.
